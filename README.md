## Collection Page

The **Collection Page** displays a list of available products and provides users with an intuitive way to browse, filter, search, and sort items.

### Features

- **Category & Subcategory Filtering**  
  Filter products based on category and subcategory (type) to quickly narrow down results.

- **Search Functionality**  
  Find specific products by entering keywords into the search bar.

- **Sorting Options** 
  Sort products by price:
  - Ascending (low → high)
  - Descending (high → low)
